1. SELECT - выбирает какие колонки песчатать

1,5. COUNT - считает кол-во элементов (Select Count(*)... - посчитает
сколько элементов в таблице)

2. * - все колонки

3. from - из какой таблицы будет печатать

4. limit - ограничение количество печатания 

4,5. offset - пишется после limit, означает сколько строк пропустит перед
лимтом (limit 1 offset 2 - выведет 1 строку (3ю) пропустив первые 2)

5. DISTINCT - пишет только неповторяющиеся элементы

6. Where(где) - условие как if (where name = 'Sofia' печатает только
тех у кого имя софия) (ВАЖНО писать в "" нельзя тк он не будет принимать
str внутри)

7. Order by колонка (asc(возрастание по умолчанию), desc(убывание)) - сортирует таблицу по колонке

1-3. Пример
Select * 
from employees

4-5. Пример
Select distinct name 
from employees
limit 5
