9.insert into название таблицы values('hello!',10); - вставляем значения в таблицу (это будет 1 строка в итоге (hello!|10))

10. primary key - это поле в SQL-таблице используется для хранения уникальных идентификаторов объектов, которые перечислены в 
таблице. Например, это может быть ID клиента или товара.
	1)Каждая запись в таком поле должна быть уникальной.
	2)Запись в поле не должна быть пустой.
	3)В одной таблице может быть только один ключ.
